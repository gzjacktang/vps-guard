# VPS Guard 项目约定

## Agent skills

### Issue tracker

需求、PRD 和实施任务使用 GitHub Issues；外部 PR 暂不作为自动分类的需求入口。详见 `docs/agents/issue-tracker.md`。

### Triage labels

使用默认五状态标签：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。详见 `docs/agents/triage-labels.md`。

### Domain docs

本项目采用单一上下文布局，领域术语位于根目录 `CONTEXT.md`，架构决策位于 `docs/adr/`。详见 `docs/agents/domain.md`。
